Урок 6. Введение в CSS

Как изменить внешний вид веб-страницы

Теория

HTML отвечает за структуру страницы, а CSS — за её внешний вид. С помощью CSS можно менять цвет, размер текста, фон и расположение элементов.

color

Цвет текста

background

Фон элемента

font-size

Размер текста

text-align

Выравнивание текста

<style>
h1 {
  color: blue;
}

p {
  font-size: 18px;
}
</style>

Практика

Добавь стили к странице:

  • Сделай заголовок синим
  • Сделай текст больше

Мини-проект

Оформи страницу о своём городе:

  • добавь цвет текста
  • измени размер шрифта
  • сделай фон страницы
