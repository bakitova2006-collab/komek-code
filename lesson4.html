Урок 4. Таблицы в HTML

Как структурировать данные с помощью таблиц

Теория

Таблицы используются для отображения структурированной информации. Например: расписание уроков, результаты соревнований, список учеников.

<table>

Создаёт таблицу

<tr>

Строка таблицы

<td>

Ячейка таблицы

<th>

Заголовок таблицы

<table border="1">

<tr>
<th>Имя</th>
<th>Город</th>
</tr>

<tr>
<td>Айжан</td>
<td>Костанай</td>
</tr>

<tr>
<td>Алия</td>
<td>Астана</td>
</tr>

</table>

Практика

Создай таблицу с тремя колонками:

  • Название города
  • Регион
  • Интересный факт

Рефлексия

Ответь для себя на вопросы:

  • Что делает тег table?
  • Чем отличается td от th?
  • Где можно использовать таблицы?
