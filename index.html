Что есть в курсе

Структурированное обучение от простого к более сложному

H

Модуль 1. HTML

Основы web-страницы, теги, текст, изображения, ссылки, таблицы и структура документа.

C

Модуль 2. CSS

Цвета, шрифты, блоки, отступы, карточки, адаптивность и современное оформление сайта.

J

Модуль 3. JavaScript

События, кнопки, логика, работа с элементами страницы и создание интерактивных элементов.
