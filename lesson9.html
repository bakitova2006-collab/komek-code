Урок 9. Flexbox

Как красиво расположить элементы на странице

Теория

Flexbox — это способ расположения элементов в строку или колонку. Он используется почти на всех современных сайтах.

display: flex

Включает flex-режим

justify-content

Выравнивание по горизонтали

align-items

Выравнивание по вертикали

gap

Расстояние между элементами

.container {
  display: flex;
  justify-content: center;
  gap: 20px;
}

Практика

Сделай карточки в ряд:

  • используй display: flex
  • добавь расстояние между элементами

Мини-проект

Создай блок карточек:

  • 3 карточки в ряд
  • одинаковый стиль
  • отступы между ними
