Введение в HTML

Что такое HTML и как создаётся первая страница

Тема урока: Основы HTML и создание первой веб-страницы

Цель урока: Познакомиться с HTML, понять структуру веб-страницы и написать свой первый HTML-код

Теория

HTML (HyperText Markup Language) — это язык разметки, с помощью которого создаются веб-страницы. Он не является языком программирования — HTML определяет структуру и содержание страницы.

Страница в HTML состоит из элементов, называемых тегами. Теги описывают различные части страницы: заголовки, абзацы, списки, изображения, ссылки и т.д.

Каждый тег имеет открывающий и закрывающий вариант: <h1>Заголовок</h1>, <p>Абзац текста</p>. Теги могут содержать атрибуты, которые уточняют их поведение или стиль.

Пример из жизни

Любая веб-страница начинается с HTML-разметки. Например, страница школы может содержать:

  • Название школы
  • Описание
  • Новости
  • Контакты и расписание занятий

HTML помогает структурировать эти элементы и правильно их отображать в браузере.
