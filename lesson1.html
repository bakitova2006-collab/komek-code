Введение в HTML

Что такое HTML и как создаётся первая страница

Теория

HTML — это язык разметки, с помощью которого создаются страницы сайтов. Он описывает структуру страницы: заголовки, текст, изображения, списки.

HTML состоит из тегов.

Пример из жизни

Любая веб-страница начинается с HTML-разметки. Например страница школы может содержать:

  • Название школы
  • Описание
  • Новости
