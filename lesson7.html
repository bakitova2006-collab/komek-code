Урок 7. Цвет, фон и текст

Как сделать сайт красивым и читаемым

Теория

CSS позволяет менять внешний вид страницы: цвет текста, фон, размер шрифта и выравнивание.

color

Цвет текста

background-color

Цвет фона

font-size

Размер текста

text-align

Выравнивание текста

h1 {
  color: red;
  text-align: center;
}

body {
  background-color: lightblue;
}

p {
  font-size: 18px;
}

Практика

Сделай страницу красивой:

  • заголовок — красный
  • фон — светло-голубой
  • текст — больше

Мини-проект

Оформи страницу о своём городе:

  • сделай красивый фон
  • измени цвет текста
  • выравни заголовок по центру
