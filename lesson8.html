Урок 8. Отступы и границы

Как управлять расстояниями между элементами

Теория

В CSS есть свойства, которые управляют расстояниями:

margin

Внешний отступ (снаружи блока)

padding

Внутренний отступ (внутри блока)

border

Граница вокруг элемента

border-radius

Скругление углов

div {
  margin: 20px;
  padding: 15px;
  border: 2px solid black;
  border-radius: 10px;
}

Пример

Без отступов элементы выглядят "слипшимися". С отступами — аккуратно и красиво.

Практика

Сделай карточку:

  • добавь отступы
  • добавь границу
  • сделай скругление

Мини-проект

  • добавь отступы
  • сделай границу
  • сделай скруглённые углы
