Урок 5. Формы в HTML

Как создавать поля ввода, кнопки и простые формы на веб-странице

Тема: ввод данных
Формат: теория → практика → мини-проект
Итог: форма регистрации
Этап 1

Теория

Основы

Что такое форма

Форма нужна для того, чтобы пользователь мог вводить данные на сайте. Например: имя, электронную почту, пароль, возраст или сообщение.

Формы используются при регистрации, входе в аккаунт, заполнении анкеты, заказе товара и отправке обратной связи.

<form>

Создаёт форму для ввода данных.

<input>

Поле ввода текста, почты, пароля и других данных.

<label>

Подпись к полю формы.

<button>

Кнопка отправки или действия.

type="text"

Обычное текстовое поле.

type="email"

Поле для электронной почты.

type="password"

Поле для пароля.

placeholder

Подсказка внутри поля.

Пример из жизни

Когда ты регистрируешься на сайте, ты заполняешь форму: вводишь имя, почту и пароль. Всё это создаётся с помощью HTML-форм. 

<form>
  <label>Имя:</label>
  <input type="text" placeholder="Введите имя">

  <label>Email:</label>
  <input type="email" placeholder="Введите email">

  <button>Отправить</button>
</form>
Этап 2

Разбор тегов

Понимание структуры
Что важно запомнить
• тег <form> объединяет все элементы формы
• тег <input> создаёт поле ввода
• атрибут type определяет вид поля
• тег <button> создаёт кнопку
• подписи к полям делают форму понятнее
Подсказка

Хорошая форма должна быть не только рабочей, но и понятной для пользователя. Поэтому подписи и логичный порядок полей очень важны.

Этап 3

Практика

Основное задание

Создай простую форму регистрации. В ней должны быть:

  • заголовок страницы
  • поле для имени
  • поле для email
  • поле для пароля
  • кнопка отправки

Мини-проект: страница с формой регистрации.

Это уже похоже на настоящий элемент сайта, поэтому такой урок хорошо завершает модуль HTML.

Идея для усложнения

Можно добавить поле возраста, комментарий или выбор любимого школьного предмета.
